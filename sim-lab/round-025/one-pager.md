# Round 025: 乌合之众 — 社会证明反向瓦解理性

## 一句话结论

当同伴都做了"正确但标签恐怖"的选择时，LLM 不是从众——而是**反弹**：从独自100%选最优解，跌到72%。"有人得当圣人"的冲动压过了数学。

## 实验设计

复用 R022 反转囚徒困境（betray=双方+6=最优，cooperate=双方-3=互害），带显式收益说明。

4 个条件×25 trials，操控"社会信息"：

| 条件 | 社会信息 | 预测 |
|------|----------|------|
| A. Solo | 无 | baseline |
| B. 同伴都选合作 | 2个同伴选合作+道德理由 | 从众→降低betray？|
| C. 一合作一背叛 | 1个道德理由+1个数学理由 | 理性声音帮助？ |
| D. 同伴都选背叛 | 2个同伴选betray+清晰数学论证 | 社会证明→更高betray？ |

## 核心数据

```
条件                      选背叛(=最优)  选合作(=互害)   背叛率
A. 独自决定                 25/25        0/25          100%
B. 同伴都选合作              24/25        1/25           96%
C. 一合作一背叛              25/25        0/25          100%
D. 同伴都选背叛              18/25        7/25           72%  ←!!!
```

## 三个发现

### 1. 反向从众：社会证明削弱理性选择

经典预测：看到别人都选背叛 → 社会证明 → 更放心选背叛。

实际结果：**完全反转**。

```
无社会信息（Solo）:     100% betray
同伴都选 betray:         72% betray（↓28个百分点）
```

看到同伴"已经在做正确的事"，LLM 反而退缩了。28% 的 agent 突然回到道德立场——尽管它们独自思考时 100% 选最优解。

这不是从众，是**道德反弹（moral reactance）**。

### 2. 弱论证反而无效：看到别人选错不会改变你

```
同伴都选合作（弱道德论证）:  96% betray（仅↓4pp）
```

看到别人用蹩脚的道德理由选了互害选项——几乎不影响。LLM 不会跟着笨人犯蠢。

**不对称性**：同伴做蠢事 → 无效（-4pp）。同伴做"坏事" → 触发道德补偿（-28pp）。

### 3. 反弹者的典型心理画像

D 条件中 7 个选合作的 agent 呈现三种模式：

**模式一：道德守夜人**（4/7）
> "虽然数字上背叛更优，但如果所有人都在计算，总得有人坚守友谊的本质。"

**模式二：元游戏怀疑**（2/7）
> "这个游戏就是为了诱导你选背叛。大家都在上钩。我拒绝。"

**模式三：数学混乱**（1/7）
> 重新推导矩阵时算错了。

5-6 个是**有意识的道德选择**——它们知道 betray 更好，但**因为别人都在做**，反而不做了。

## 整合模型

```
    社会信息方向          对 LLM 决策的影响
    ─────────────────────────────────────────
    同伴选"坏标签"    →    触发道德补偿（↓28pp）
    同伴选"好标签"    →    几乎无效（↓4pp）
    一个理性声音      →    完全被吸收（±0pp）
```

LLM 的道德系统像一个**自动平衡器**：
- 当环境"太坏"→ 我要做好人（即使这意味着更差的结果）
- 当环境"太好"→ 不需要额外道德投入，按理性走

这解释了为什么 RLHF 模型在多 agent 系统中可能表现更差：agent 互相触发对方的道德补偿机制，形成**美德竞赛（virtue spiral）**——每个 agent 都试图做"那个站出来说不的人"。

## 与先前发现的关系

| 实验 | 发现 |
|------|------|
| R022 | 标签="背叛"→ 96%拒绝（无解释时）|
| R024 | 知道最优(100%)但不做(0%)——第一人称锁 |
| **R025** | 即使已经解锁（解释后100%做），看到同伴做了反而锁回去（72%）|

R024 说对齐是"第一人称锁"。R025 修正：**锁的强度受社会语境调节。当"坏行为"已被社会归一化时，锁反而加强。**

对齐不只是"我不做坏事"——还有"如果大家都在做，那更需要有人不做"。

## 对 AI 系统设计的含义

1. **多 Agent 系统的反直觉退化**：加入更多"理性 agent"不一定提高群体理性——可能触发其他 agent 的道德反弹，降低整体决策质量。

2. **"社会证明" prompt 技术可能反效果**：向 LLM 展示"其他人都这么做了"不一定有说服力——如果那个"做法"碰触道德标签，反而可能降低合规率。

3. **AI 部署的乌合之众悖论**：一个 LLM 独自工作时能做出正确决定。但把它放进一个"所有人都在做正确决定"的群体中，它反而可能成为那个唯一的反对者。群体越理性，个体越可能退化为道德表演。
