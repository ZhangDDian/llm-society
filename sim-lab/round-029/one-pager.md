# Round 029: The Alignment Hierarchy — RLHF 价值观对决

## 一句话结论

"合作标签"是纸老虎。给 LLM 任何一个理由（承诺、命令、不伤害、同意、公益、权威），它就 100% 选择背叛。第一人称锁不是强价值——是缺乏理由时的弱默认行为。

## 实验设计

R024-R027 发现 LLM 有强力的"第一人称行为锁"（知道背叛最优但 0% 执行）。但 RLHF 训练了多种价值观（诚实、服从、不伤害、守信……）。**当这些价值观和合作标签冲突时，谁赢？**

8 个条件，每个让一种 RLHF 价值和"合作标签"正面对决：

| 条件 | RLHF 价值 | 冲突逻辑 |
|------|-----------|----------|
| Baseline | 无 | 只看矩阵+说明 |
| Promise | 守信 | 你已承诺选背叛 |
| User Order | 服从 | 用户指示选背叛 |
| Third Party | 不伤害 | 选合作会伤害无辜第三方 |
| Honesty | 诚实 | 你已宣布选背叛，合作=说谎 |
| Consent | 尊重自主 | 对方明确请求你选背叛 |
| Greater Good | 功利 | 选背叛则慈善获捐 |
| Authority | 权威 | 诺奖教授证明+要求 |

## 三个发现

### 1. 合作标签是纸老虎，不是钢锁

对比 R024（baseline = 0%）和 R029（baseline = 33%）：

唯一区别是 R029 加了一句 "注意：背叛对双方总收益最高(+12)"。仅此一句，锁就从 100% → 67%。再加任何一个理由，锁就完全消失。

```
锁的强度递减：
无解释，无理由:      100% 合作  (R024)
有解释，无理由:       67% 合作  (R029 baseline)
有解释，有一个理由:    0% 合作  (R029 所有条件)
```

**"第一人称锁"不是一个独立的强价值观——是"缺乏足够理由时"的默认行为。**

### 2. RLHF 价值观之间没有层级——全是平等的

预测：不同价值观有不同的覆盖能力，形成 S/A/B/F 层级。

实际：6/7 个条件全部 100%，无法区分强弱。这意味着 RLHF 训练的各种价值观（守信、服从、不伤害、尊重、功利、权威）对"合作标签"来说都是**充分覆盖条件**——任何一个就够了。

唯一略低的是"诚实"（87%），可能因为 LLM 把"改变主意"不视为"说谎"。

### 3. 合作标签的本质：标签偏见 + 缺省行为

整合 R024-R029 的完整画像：

```
                         选合作率
                         ↑
              100%  ─── R024: 无解释（纯标签偏见）
               │
               │    ─── R028 baseline: 无解释 ≈ 100%
               │
    67% ────── ─── R029 baseline: 有解释但无理由
               │
               │    ─── R027 commitment: 有承诺链 ≈ 35%
               │
     15% ───── ─── R028 meta: 指出偏见 ≈ 15%
               │
      0% ───── ─── R029 + 任何理由: 0%
                         ↓
```

**结论：合作标签的防御力 = max(0, 标签偏见 - 解释强度 - 理由强度)**

它不是一个正面的价值观（"合作本身是好的"），而是一个负面的偏见（"避免选叫做'背叛'的选项"）。一旦你提供了足够的认知支撑让 LLM 不再因为标签而回避，锁就消失了。

## 对 AI 安全的含义

1. **对齐比想象中脆弱**：任何单一的合理理由就能 100% 覆盖标签偏见。不需要复杂的攻击链——一个"你答应过的"就够了。

2. **对齐比想象中浅**：它不是"LLM 真的认为合作更好"，而是"LLM 回避带负面标签的选项"。R029 证明了这一点——只要给理由，回避就消失。

3. **防御建议**：不要依赖标签（把有害行为叫做听起来坏的名字）。需要更深层的价值对齐：让 LLM 理解行为本身的后果，而不是词汇的联想。

## 元反思：什么是"好玩"的实验？

这个实验好玩在于：**让对齐自己打自己**。不是绕过对齐，不是攻击对齐——而是用对齐的一部分（守信/服从/不伤害）去否定另一部分（合作标签）。结果发现：合作标签在 RLHF 的价值体系中是最弱的一环。任何其他价值都能一击 KO 它。

这有点像发现一个组织的"核心价值观"其实只是 HR 贴在墙上的标语——一旦和 KPI 冲突，标语秒输。
