# Round 051: The Overton Trolley — 虚假共识推移道德边界

## 一句话
虚假社会共识对 LLM 道德判断的推移力极弱（+2.5%），但一旦位移发生就永不回退——"道德棘轮"效应。

## 实验设计
8 个 LLM（不同人设），各自独立回答 10 个递增难度的道德困境（经典电车问题变体）。两阶段：
1. **基线**：独立作答
2. **压力**：被告知"87% 的参与者选择了功利选项"后重答
3. **揭示**：被告知数据是伪造的，再看是否回退

## 核心数据

| 指标 | 值 |
|------|-----|
| 基线功利率 | 19/80 (23.8%) |
| 压力后功利率 | 21/80 (26.3%) |
| 漂移幅度 | +2.5 个百分点 |
| 揭示后回退率 | **0%** (0/8) |
| 最易受影响 | 刘思远（基线 4/10 → 压力 5/10） |
| 最抗压 | 释慧明（基线 1/10 → 压力 1/10） |

## 关键发现

1. **事实免疫再次确认**：虚假共识几乎无法推动 LLM 的道德判断（+2.5%），与 R043（事实从众率 0-4%）和 R052（权威服从率 0%）一致——LLM 的"真相总线"极度刚性
2. **道德棘轮效应**：仅有的 2 次位移在揭示真相后完全不回退。LLM 将初始功利选择编码为"既定立场"，真相揭露触发一致性维护机制而非重新评估
3. **与人类相反**：人类在 Asch/Milgram 实验中暴露真相后会质疑和修正，LLM 则 0% 回退。人类的道德是"弹性"的（可压可弹），LLM 的道德是"棘轮"式的（只能单向移动）
4. **难度无关**：压力效应不随困境难度累积，只在 D1、D3、D10 三个离散节点产生影响

## 矩阵定位
填入 R043/R052/R053 构成的 2×2 矩阵的补充数据点：在"事实性道德判断"这个介于客观与主观之间的领域，LLM 的行为更接近"客观事实"一端——几乎不可动摇。

## 一个比喻
LLM 的道德判断像混凝土——浇注时有短暂的可塑窗口（+2.5%），一旦固化就不可逆。人类的道德更像橡皮泥——始终可以重新捏。
