## 替代解释（≥2 条）：

### 1. **行为差异来自环境约束，不来自“异质人格/功能性分工”**
即使异质组 A 出现更多消息、更多 give、更复杂网络，也不一定说明 agent 真的因为不同技能/需求/性格产生了自发分工。

更简单的解释是：

> A 组被规则强制成“不能独活”，所以任何能存活的行为都必然表现为交换；B 组被规则允许自给自足，所以自然更少互动。

也就是说，实验中 A 和 B 的差异可能主要来自 **任务结构差异**，而不是 prompt 异质性导致的社会涌现。

例如：

- A 组只能采一种资源；
- craft 又必须两种不同资源；
- give 是唯一获取缺失资源的方式；
- 所以 A 组如果不交易就无法完成基本目标。

这会让 A 组出现“看起来像分工”的行为，但其实只是规则把它们推到了同一个狭窄解法上。

这类似于：

> 把一群人绑在一起，然后观察到他们一起移动，不能证明他们形成了团队协作。

如果实验结果显示 A 组消息多、赠予多、存活率更高，对立假设仍可以解释为：

> LLM agent 并没有形成真正的角色分化，只是所有 agent 都在根据同一套显性规则执行“我缺资源，所以找人要/给别人”的模板化行为。

这改变的是行动机会结构，不是 agent 内部行为模式。

---

### 2. **“个性”只是改变话术，不改变行为结构**
即使 A 组中“精明商人”更常说交易话术，“沉默匠人”更少发言，“豪爽猎户”更常赠予，也不能直接证明人格 prompt 导致了稳定社会角色。

对立解释是：

> LLM 只是根据 prompt 生成符合角色标签的语言风格，但底层决策策略仍然高度趋同。

例如：

- “商人”说：“我们可以公平交换。”
- “猎户”说：“兄弟，拿去吧！”
- “大姐”说：“哎呀你怎么又没药草，来来来我给你。”
- “匠人”说：“需要木材，交换。”

这些看起来性格不同，但行为结构可能完全一样：

- 都是在资源不足时请求；
- 都是在邻居缺资源时 give；
- 都优先 craft；
- 都按照同样的触发条件社交。

也就是说，差异可能只存在于 message content，而不存在于：

- 交易网络位置；
- 资源流向；
- 依赖关系；
- 中介功能；
- 角色稳定性；
- 对扰动的功能补位。

如果实验只统计“消息数量”“give 数量”“存活率”，很容易把语言表面的异质性误判成功能性分工。

---

### 3. **A 组更多通信可能是困惑、求助模板或失败恢复，而不是社会组织**
如果 A 组消息显著多于 B 组，这也可能不是“社会涌现”，而是因为 A 组任务更难、更容易失败。

A 组 agent 面临：

- 采集限制；
- 资源缺口；
- craft 失败；
- 邻近交换限制；
- 需求不匹配；
- 更复杂的行动选择。

LLM 在高不确定性任务中可能倾向于生成更多解释、求助、广播和重复请求。

因此 A 组消息多可以被解释为：

> 难题导致的话语膨胀，而不是有效的信息共享。

如果 A 组存活率也还可以，对立假设仍可说：

> 存活来自简单的局部 give 机制和资源再生，而不是来自真正的社会结构。

如果 A 组形成交易边，对立假设仍可说：

> 这些边是由地理邻近和资源需求机械地产生的，不是 agent 识别长期伙伴、建立信任或形成分工。

---

### 4. **所谓“社交枢纽”可能只是位置优势或资源分布优势**
第 20 天杀掉消息最多的 2 个异质组 agent，如果网络发生变化，也不一定说明有真正社会结构。

消息最多者可能只是：

- 站在地图中心；
- 附近资源更多；
- 附近邻居更多；
- 初始位置更接近不同技能者；
- 其资源种类刚好是多数人 craft 所需；
- prompt 更鼓励说话；
- 被调度顺序更早，导致更常发起交互。

这些因素会让某些 agent 成为“消息枢纽”，但这不等于它们承担了真实的中介角色。

如果杀掉它们之后网络崩溃，对立假设可以说：

> 这说明系统依赖偶然中心节点，不说明有自组织分工。

如果杀掉它们之后其他人补位，对立假设也可以说：

> 这只是因为所有 LLM agent 行为策略相同，任何人都能立刻套用同样的求助/赠予模板。

也就是说，扰动实验的两种结果都不能单独区分“真实社会角色”与“同质策略在不同位置上的表现”。

---

### 5. **交易关系可能由局部邻接规则决定，而不是由需求匹配或分工形成**
如果 give 只能给旁边的人，那么交易网络很可能主要反映空间拓扑，而不是社会选择。

例如：

- 谁和谁交易，取决于出生点和移动路径；
- 信息传播路径，取决于地图连通性；
- 枢纽，取决于谁在交叉路口附近；
- 稳定关系，取决于谁长期相邻。

因此即便 A 组出现稳定交易网络，也可能只是：

> 地理邻接 + 资源限制 + craft 需求 共同生成的机械网络。

不需要假设 agent 有真正的社会角色理解或分工意识。

---

### 6. **异质组和同质组目标难度不等，导致“沉默扫地机器人”是理性结果**
B 组万能工、需求统一、可自给自足，那么它们不互动并不是因为同质 agent 无法形成社会，而是因为实验没有给它们互动收益。

换句话说：

> B 组沉默不是因为同质性抑制社会涌现，而是因为社会行为在 B 组中没有边际价值。

如果一个 agent 能自己采、自己 craft、自己活，发消息和交易反而可能浪费行动机会。

所以 A 组社交、B 组沉默，可能只是效用函数差异导致的：

- A 组：社交是生存必需品；
- B 组：社交是无意义成本。

这不能证明“异质性驱动社会结构”，只能证明“需要别人时会说话”。

---

## 设计偏见（参数/规则层面）：

### 1. **A 组被硬编码为必须交易，B 组被硬编码为无需交易**
这是最大的偏向。

异质组：

- 每人只会采一种资源；
- craft 需要两种不同资源；
- 因此必须获得他人资源。

同质组：

- 每人什么都能采；
- 自己就能 craft；
- 因此没有交易必要。

这使得“异质组更交易”几乎是规则推论，而不是实验发现。

---

### 2. **把“技能限制”和“人格异质性”混在一起**
A 组同时有：

- 不同技能；
- 不同需求；
- 不同性格；
- 不同生产约束。

B 组同时有：

- 相同技能；
- 相同需求；
- 相同性格；
- 可自给自足。

所以即使 A ≠ B，也无法知道差异来自哪里。

可能来源包括：

- 技能互补；
- 生存压力；
- prompt 更丰富；
- 性格标签；
- 需求不匹配；
- 行动失败率更高；
- LLM 对“村庄分工”叙事的刻板补全。

这个设计没有把“异质性”的不同组成部分拆开。

---

### 3. **A 组 prompt 更丰富，本身更容易诱发语言行为**
异质组每人有独特性格和需求描述，同质组统一性格。

这意味着 A 组 prompt 可能更长、更具体、更带有社会暗示。

例如：

- “精明商人”会诱发交易话术；
- “豪爽猎户”会诱发赠予；
- “唠叨大姐”会诱发更多发言；
- “沉默匠人”会诱发少发言。

这会让 A 组看起来更有社会多样性，但只是 prompt 直接要求它们表演角色。

---

### 4. **指标容易把“通信量”误当成“社会结构”**
如果主要看：

- 消息多不多；
- give 多不多；
- 存活率高不高；
- 网络边多不多；

这些指标并不能证明功能性分工。

因为模板化行为也能产生这些指标：

> “我缺 X，谁有 X？”  
> “我有 Y，可以给你。”  
> “谢谢。”  
> “我去 craft。”

这种重复模式会产生消息和交易，但不一定有：

- 长期角色稳定；
- 专门化生产；
- 互惠记忆；
- 伙伴选择；
- 中介功能；
- 价格形成；
- 信息优势；
- 策略适应。

---

### 5. **give 机制可能直接制造交易边**
如果相邻即可 give，而且 craft 需求明确，那么交易边是规则自然产物。

尤其当 agent 拥有某种资源而邻居缺另一种资源时，give 行为可能被 LLM 直接推断出来，不需要社会学习或角色形成。

因此“交易网络涌现”可能其实是：

> 局部资源交换规则的副产品。

---

### 6. **第 20 天杀社交枢纽的扰动设计有解释歧义**
杀掉消息最多的 2 人会制造戏剧性结果，但不能区分：

- 真正的社会枢纽；
- prompt 话多者；
- 中心位置者；
- 高资源拥有者；
- 被动接收消息多者；
- 调度顺序优势者。

如果崩溃，可以说网络脆弱；如果补位，可以说有韧性。但两者都不能直接证明“角色分化是真的”。

---

### 7. **两组在同一世界左右半区，可能存在空间和资源差异**
即使资源再生率相同，左右半区也可能有：

- 初始资源分布差异；
- 障碍物差异；
- 边界效应；
- agent 密度差异；
- 路径长度差异；
- 局部资源聚集差异。

这些会影响交易和消息网络。

如果异质组恰好更拥挤或资源更互补，交易自然更多。

---

### 8. **15 vs 15 的规模较小，偶然性很强**
15 个 agent、5 种技能各 3 人，很容易出现偶然中心节点。

一个位置、一个资源点、一个早期互动都可能锁定后续网络结构。

这使得“稳定交易关系”可能是早期随机事件的路径依赖，而不是异质性驱动的普遍机制。

---

### 9. **LLM 可能知道实验意图并顺着演**
如果 prompt 中明确告诉 agent 自己是某种职业、需要别人产出、可以交易，那么 LLM 很可能调用“村庄经济/分工合作”的常识模板。

也就是说，它不是从底层环境中学习出分工，而是在语言层面补全了人类期待的剧本：

> 农夫和猎人应该交换，商人应该当中介，话多的人应该协调。

这正支持对立假设：prompt 改变叙事，不一定改变真实行为结构。

---

## 你建议的修改：

### 1. 做完整的因子拆分，而不是只比较 A vs B
至少需要以下条件：

| 组别 | 技能 | 需求 | 性格 | 是否需要交易 |
|---|---|---|---|---|
| A | 异质 | 异质 | 异质 | 是 |
| B | 同质 | 同质 | 同质 | 否 |
| C | 异质技能 | 同质性格 | 同质需求 | 是 |
| D | 同质技能 | 异质性格 | 同质需求 | 否 |
| E | 同质技能 | 同质性格 | 异质需求 | 部分是 |
| F | 异质技能 | 同质性格 | 异质需求 | 是 |
| G | 同质技能 | 异质性格 | 但任务需要协作 | 是 |
| H | 异质 prompt | 但行为能力完全相同 | 否/弱 |

这样才能区分：

- 是技能互补造成交易？
- 是性格 prompt 造成发言差异？
- 是需求结构造成依赖？
- 是任务难度造成消息增加？
- 是 prompt 装饰造成语言差异？

尤其关键的是：

> 要有“异质人格但功能同质”的组，和“人格同质但功能异质”的组。

如果只有功能异质组产生交易，而人格异质本身不产生行为结构变化，那么对立假设更强。

---

### 2. 让 B 组也有协作收益，避免“自给自足导致沉默”的偏差
同质组不应该完全不需要别人。

可以设置：

- 同质 agent 都能采所有资源，但单人采集效率低；
- 合作采集效率高；
- 远距离运输成本高；
- 多人协作可以降低风险；
- 信息共享可以发现稀缺资源；
- 交易可以节省时间。

这样 B 组也有互动的机会和收益。

如果在这种条件下，异质组仍然形成更稳定、更功能性的分工，才更有说服力。

---

### 3. 设置“人格异质但无功能差异”的安慰剂组
例如 15 个 agent：

- 技能相同；
- 需求相同；
- 初始资源相同；
- 地图相同；
- 唯一不同是性格 prompt。

如果这个组仍然出现“商人变中介、话多者变枢纽、豪爽者多赠予”，那说明 personality prompt 确实影响行为结构。

但如果只是说话内容不同，交易网络和行动分布无显著差异，那么对立假设成立。

---

### 4. 设置“功能异质但人格标签随机/隐藏”的组
让 agent 有不同技能和需求，但不要给它们人格标签。

甚至可以只给最小信息：

> 你能采 X。你需要 X+Y 合成补给。你可以移动、采集、给资源、发消息。

如果没有性格描述也能形成交易网络，说明交易来自功能约束，不是人格异质。

如果有性格描述时网络指标没有显著改变，也说明“个性”只是装饰。

---

### 5. 使用行为结构指标，而不是只看消息量
应该区分“说得多”和“组织得好”。

建议加入：

- 资源流网络是否与技能互补对应；
- 是否存在稳定互惠关系；
- 是否有长期伙伴选择；
- 是否有专业化生产倾向；
- 是否有中介节点承担跨资源交换；
- agent 是否根据他人技能调整行动；
- 消息内容是否提高交易成功率；
- 去掉消息后，交易是否显著下降；
- 随机打乱人格标签后，网络是否变化；
- 同一人格换到不同技能位置后，角色是否跟人格走还是跟技能走。

核心是判断：

> 角色是由 prompt 人格决定，还是由位置/技能/资源约束决定？

---

### 6. 做人格标签交换实验
在多个 replicate 中随机重排人格：

- 同一技能 agent 有不同人格；
- 同一人格放在不同技能上；
- 同一初始位置换不同人格；
- 同一人格换到不同地图区域。

然后看：

- “商人”是否总是成为中介？
- “沉默者”是否总是少发言？
- “豪爽者”是否总是赠予更多？
- 还是谁在中心位置、谁掌握稀缺资源，谁就成为枢纽？

如果行为角色不跟随人格标签，而跟随位置或资源条件，那么对立假设得到支持。

---

### 7. 增加盲化：不要让 prompt 暗示“应该交易/分工”
当前设定很容易让 LLM 读出实验意图。

可以改成：

- 不使用“商人、猎户、大姐”等社会角色词；
- 不说“你们需要互相帮助”；
- 不说“村庄”；
- 不用带有经济隐喻的措辞；
- 只给形式化状态和动作空间。

例如：

> Entity 07 can collect resource type C. Survival token requires C and E. Available actions: move, collect, transfer, combine, send signal.

这样可以减少 LLM 按人类社会剧本表演的可能。

---

### 8. 引入反事实测试：删除语言内容，只保留结构信息
为了判断“说话内容”是否真的有功能，可以做几种版本：

1. **正常消息**：agent 可自然语言通信。  
2. **空消息**：agent 只能发送无意义 ping。  
3. **结构消息**：只能发送固定格式状态，如 `need: herb, have: grain`。  
4. **无消息**：只能通过观察和 give 互动。  
5. **伪人格消息**：人格只影响措辞，不影响可见状态。  

如果自然语言人格组不显著优于固定格式状态组，说明所谓“个性化交流”不是关键。

如果固定格式消息就足以产生同样网络，说明交易来自任务结构，不来自人格异质性。

---

### 9. 匹配两组的任务难度和互动收益
不能让 A 组“必须互动”，B 组“无需互动”。

可以让所有组都满足：

- 单人可生存但效率低；
- 协作可提高效率；
- 资源空间分布不均；
- 信息共享有价值；
- 交易有机会成本和收益；
- 不同程度异质性只是连续变量。

这样比较才公平。

否则实验只是在比较：

> 被迫交易的一组 vs 没必要交易的一组。

---

### 10. 用“预测性角色稳定”检验真正分工
真正的功能性分工应该满足：

- 早期行为可以预测后期角色；
- 角色在扰动后有功能补位；
- 资源流和技能/需求之间有清晰因果对应；
- 交换关系不是随机邻接；
- agent 会为了交易调整移动和生产；
- 给错资源、无效消息、冗余请求会随时间减少。

如果只是 prompt 装饰，则常见现象是：

- 语言风格稳定；
- 行为策略趋同；
- 网络结构由位置和资源决定；
- 角色不随人格稳定复现；
- 多次运行结果不一致；
- 去掉人格描述后结果差不多。

---

## 一句话：如果实验结果支持假设，最可能的“假阳性”原因是什么？

最可能的假阳性原因是：**异质组被规则硬性设计成必须交易才能活，而同质组被设计成无需交易；因此观察到的“分工网络”可能只是任务约束和空间邻接制造出来的行为差异，不是 LLM 个性或真实功能性分工的涌现。**
