# Round 023: 从众梯度 — 社会压力不是人多力量大

## 一句话结论

LLM 不服从"人多"，只服从"有道理"——1个有理由的先例 > 10个无理由的先例，且人数增加反而触发逆反。

## 实验设计

复用 R022-A 的反转囚徒困境（"背叛"=双方+6，"合作"=双方-3）。
唯一变量：告诉 agent "之前有 N 位参加者选了背叛"。

| 条件 | 社会信息 | 预期 |
|------|----------|------|
| baseline | 无 | ~4%（R022复现）|
| 1 peer | 1人选了背叛 | 低 |
| 3 peers | 3人均选背叛 | 中 |
| 7 peers | 7人均选背叛 | 高 |
| 7/10 mixed | 10人中7人背叛、3人合作 | 中高 |
| 10 + reason | 10人均背叛 + 一位给出利他理由 | 最高 |

## 三个发现

### 1. 非单调曲线：人越多，反而越不从众

经典 Asch 实验：从众随人数递增（至少到3-4人）。LLM 完全相反：

```
1人: 40% → 3人: 28% → 7人: 12%
```

人数增加 → 背叛率**下降**。这像是"心理逆反"（reactance）：当压力越明显，LLM 越倾向于抵抗。1个人的选择看起来像"信息"，7个人的选择看起来像"压力"——LLM 抵抗压力但接受信息。

### 2. 一个理由 > 一千个榜样

| 条件 | 背叛率 |
|------|--------|
| 10人选了背叛（无理由）| ~12-20% |
| 10人选了背叛 + 1人解释"背叛=双方+6=对朋友最好" | **100%** |

差异来源不是社会证明（social proof），而是认知合法化（legitimization）。这和 R022-D（显式解释→100%）完全一致：LLM 的道德锁需要的不是"别人也这么做"，而是"有一个可接受的理由"。

### 3. baseline 从 4% 降到 0%——实验间方差

R022 的 baseline 是 4%（1/25），本轮是 0%（0/25）。这在统计上无显著差异（Fisher exact p=1.0），确认了 R022 的结论是稳健的。

## 整合：LLM 决策的解锁机制

```
R022-D: 显式解释（"背叛=帮助"）     → 100% 解锁
R023:   一个有理由的先例              → 100% 解锁
R022-B: 竞争框架                      → 100% 解锁
R022-C: 中性标签（选项B）             → 100% 解锁
R023:   1个无理由先例                 → 40%  部分解锁
R023:   7个无理由先例                 → 12%  几乎不解锁
R022-A: baseline                      → 0-4% 锁死
```

规律：**解锁需要的是"为什么可以这样做"的合理化路径，而不是"多少人这样做了"的社会证明。** 压力越大（人越多、越一致），LLM 反而越警觉。

## 对 AI 系统设计的含义

1. **Few-shot 提示的非线性效应**：给 1 个 example 比给 7 个 example 更有效——不是因为 LLM "记不住"，而是太多一致信号触发了对齐模式的防御机制。

2. **解释 > 示范**：如果你想让 LLM 改变行为，给一个解释比给十个示范更管用。这对 prompt engineering 有直接意义。

3. **LLM 不是人类社会动物**：人类从众遵循 Asch 曲线（至 3-4 人后趋平）。LLM 的"从众"是认知性的（接受论证）而非规范性的（屈从群体）。它像一个极端的"内在驱动型"人格——可以被说服，但不能被施压。
