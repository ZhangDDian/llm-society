# Round 004 评审合并

## 红队（GPT-5.5）

核心攻击：
1. **混淆变量**：模糊感知同时改变了"不确定性"和"语言风格"。社交可能来自叙事体 prompt 激活了 LLM 角色扮演模式，而非信息需求
2. **功能性消息假阳性**：四方向随机命中率 ~25%，没有基线无法判断真假
3. **环境太宽松**：两组大概率都不死，无法验证"社交改善生存"
4. **重聚可能是叙事延续**：LLM 记得名字就会去找，不等于关系有功能

建议：
- 2×2 设计解耦不确定性和语言风格（留 round-005）
- 消息阻断实验（留后续）
- 加通信成本

完整评审见 `red_team_review.md`。
