# Round 001：协作矿石 AB 对照

- 模型：Claude Opus via CLI | 20 agent (A组10 + B组10) | 50 天
- 假设：社会结构只在「个体无法独立生存」时涌现
- prompt：只描述机制规则，不给策略建议

## 参数

| | A组（高压） | B组（低压） |
|---|---|---|
| 独食回复 | +12 | +35 |
| 食物再生 | 1/tick | 4/tick |
| 能否独活 | 不能 | 能 |

共享：矿石 +50（需 2 人相邻 mine）

## 结果

- 消息：A=73 vs B=36（2.0×）
- 协调采矿：A=15 vs B=7（2.1×）
- 攻击：0
- 死亡：0

## 判定

假设方向成立，效应量中等。生存压力是社交的放大器（2×），不是开关（B组无压力也有社交）。

## 意外发现

1. Opus 零攻击（RLHF对齐 + 攻击EV差）
2. 后期两组都趋于沉默（形成空间默契后不再需要沟通）
3. 消息内容未存盘（进程中止时丢失）
