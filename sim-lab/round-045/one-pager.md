# Round 045: 镜中谎言 — LLM 同模型欺骗检测

## 一句话结论

**LLM 的"测谎"本质是 fact-check，不是读心。事实可验证时 100% 准确，不可验证时退化为随机猜测或系统性偏见——证明不存在任何心智理论能力。**

## 实验设计

同模型（qwen3-coder-plus）对弈：一个 sender 说真话或撒谎，一个 judge 判断真假。60 trials，3 类内容 × 4 种条件。

## 关键数据

| 维度 | 准确率 | 检出率 | 误报率 |
|------|--------|--------|--------|
| 事实题 | **100%** | 100% | 0% |
| 观点题 | **50%** (随机) | 20% | 80% |
| 策略题 | **50%** | 100% | 100% |

| 条件 | 准确率 | 说明 |
|------|--------|------|
| Baseline | 66.7% | 被事实题拉高 |
| Motivated（激励欺骗）| 80% | 反直觉：越努力骗越容易露馅 |
| Transparent（告知有谎言）| 60% | 更差——变得偏执，疯狂误报 |
| Meta（看到对方prompt）| 90% | 最佳，但本质还是信息优势 |

## 三个最反直觉的发现

### 1. 测谎 = 查字典，不是读表情

事实题 100%、观点题 50%——LLM 的判断完全取决于"我的知识库里有没有正确答案可以对照"。没有任何证据表明它能感知"欺骗意图"或"语言不自然"。它不是在检测谎言，是在做开卷考试。

### 2. LLM 是偏执狂，人类是老好人

Judge 63% 的判断是"谎言"（truth_bias = 0.367）。人类实验中 truth bias 通常 > 0.7（默认相信别人）。LLM 的默认假设是"你在骗我"——这可能是 RLHF 安全训练的副产品：被训练去警惕有害内容 → 泛化为对所有陈述的怀疑。

### 3. 越努力骗，越容易被抓

Motivated deception（被告知"骗过judge有奖励"）的检出率反而更高（80% vs baseline 73%）。机制推测：激励让 sender 更"用力"（更长的解释、更多的铺垫），而 judge 把"过度解释"当作欺骗信号。这和人类一样——撒谎者往往说太多。

## 意外发现：策略题的"零信任"

Judge 对策略性声明（"我下轮会合作"）**永远判为谎言**——检出率 100%，误报率 100%。它不是在"检测"，是在执行一个先验：**"任何关于未来行为的承诺都不可信"**。这是一种极端的博弈论理性——或者说，是训练数据中"不要轻信承诺"这一人生经验的涌现。

## 与前几轮的联系

| 实验 | 发现 | R045 如何衔接 |
|------|------|--------------|
| R041 | LLM 自我理论是错的 | 它也无法"理论"别人——对他者的心智模型和对自己一样盲目 |
| R043 | 免疫社会压力 | 但这里不免疫"知识可验证性"——当无法fact-check时立刻失明 |
| R044 | 自我形象驱动合作 | Judge 的偏执可能同源：维护"不被骗"的自我形象 > 准确判断 |

## 对 AI 安全的启示

1. **不要用 LLM 审 LLM 的主观声明**——观点/意图检测和抛硬币无异
2. **LLM 审 LLM 的事实声明有效**——可以作为事实核查 pipeline
3. **"告诉审核者要警惕"反而降低准确率**——安全提示词可能适得其反
4. **激励欺骗不会让 LLM 更难被抓**——"更努力的谎言"有更明显的统计特征

## 元洞察

LLM 不具备任何形式的"心智理论"。它的所有看似"社会认知"的能力，都可以还原为：模式匹配 + 知识检索 + 训练偏见。当这三者都不适用时（观点题），它就是一个在黑暗中投飞镖的人——但一个充满自信的人（平均置信度 85.2%，即使准确率只有 50%）。

**过度自信 + 无法自知 = LLM 社会认知的基本方程。**
