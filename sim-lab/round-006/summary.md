# Round 006 — 双人协作涌现（30人, 6技能×5, cocraft机制）

## 假设
协作门槛（两人同时在场+互补材料）驱动稳定配对自发涌现。

## 结果
| 指标 | 值 |
|---|---|
| Ticks | 60 |
| 消息总量 | 532 |
| Co-craft | 11 次 |
| 赠予 | 5 次 |
| 存活 | 6/30 (20%) |
| 网络密度 | 0.057 |
| 稳定配对(10tick) | 10 |

## 关键发现
1. **Co-craft 确实自发涌现**：11 次协作，乙组(药草)占 64%，是最受欢迎的合作对象
2. **转化率极低**：532 条消息 → 11 次协作 = 2.1%，绝大多数沟通未转化为行动
3. **网络脆弱**：tick 40 扰动后消息率从 11.9 降至 2.8，协作率从 0.25 降至 0.05，永未恢复
4. **三重门槛过高**：物理邻接 + 互补材料 + 意图对齐叠加，协调成本吞噬生存能量

## 教训
协作机制能催生社交（532 条消息远超无协作实验），但门槛设计过高时系统脆弱——少数枢纽断裂即全网瘫痪。
