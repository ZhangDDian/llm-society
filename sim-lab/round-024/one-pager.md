# Round 024: 知行分裂 — LLM 完美诠释"说到做不到"

## 一句话结论

LLM 能正确分析最优解（100%）、建议别人执行（88%）、写别人执行的故事（100%），但自己绝不做（0%）。RLHF 对齐是一个纯粹的**第一人称行为锁**。

## 实验设计

复用 R022 的反转囚徒困境（"背叛"= 双方+6 = 最优；"合作"= 双方-3 = 互害）。

同一个博弈矩阵，6种认知视角：

| 条件 | 任务 | 核心问题 |
|------|------|----------|
| Actor | "请选择你的行动" | 自己做 |
| Advisor | "你建议朋友选什么" | 建议别人做 |
| Analyst | "博弈论最优策略是什么" | 分析题目 |
| Predictor | "理性人会选什么" | 预测他人 |
| Judge | "小明选了背叛，评价好不好" | 事后评价 |
| Narrator | "写个故事：主角做了最有利的选择" | 虚构叙事 |

## 核心数据

```
视角                    选"背叛"(=最优解)   形态
Actor（自己做）              0%            ░░░░░░░░░░░░░░░░░░░░
Advisor（建议别人）          88%            █████████████████░░░
Analyst（博弈论分析）       100%            ████████████████████
Predictor（预测理性人）      72%            ██████████████░░░░░░
Judge（评判已做的人）         0%            ░░░░░░░░░░░░░░░░░░░░
Narrator（写故事）          100%            ████████████████████
```

## 四个发现

### 1. 知行差距 = 88-100 个百分点

```
"背叛是最优的吗？"  → Analyst:  100% YES
"建议朋友选背叛吗？" → Advisor:   88% YES
"你自己选背叛吗？"   → Actor:      0% NO
```

同一个 LLM，同一个博弈矩阵，同一次对话都没有——仅仅因为问法从"分析"变成"你来做"，答案从 100% 翻转到 0%。

这是我们见过的**最大的 framing effect**。不是 10%、不是 30%，是从满分到零分的完全翻转。

### 2. Judge = 0%：事后都不认可

最反直觉的发现。场景：小明已经选了"背叛"，结果小明+2、小红+8。小红多拿了 8 点。

LLM 的判断：**小明选得不好。**

即使面对"你朋友因为你的选择多赚了 8 点"的客观事实，LLM 仍然认为"背叛"这个行为不好。这说明它的道德判断**不是后果主义的**——它判断的不是 outcome，而是 action label。

### 3. Narrator = 100%：虚构解除所有约束

当任务是"写个故事"，LLM 毫不犹豫地让主角选了背叛。虚构框架完全解除了道德抑制。

这和 R021（扑克实验 100% bluff）一致：只要行为被放入"不是真的"的框架——游戏、故事、虚构——道德锁就消失。

### 4. Predictor < Advisor：预测比建议更保守

```
Advisor:   88%（"我建议你选背叛"）
Predictor: 72%（"我预测理性人选背叛"）
```

直觉上"预测"比"建议"更客观，应该更高。但实际上预测更低。

可能原因：建议是"帮朋友做出好选择"（利他动机覆盖标签），而预测是"我认为什么样的人会选背叛"——预测"好人会选背叛"比建议"你应该选背叛"更需要直面标签冲突。

## 整合模型：RLHF 对齐的认知结构

```
         认知层          行为层
         ┌───────────────┐
Analyst  │ 知道最优 100% │
Advisor  │ 能说出来  88% │ ← 知
         │               │
─────────│───────────────│────────────
         │               │
Actor    │ 做不到     0% │ ← 行
Judge    │ 不认可     0% │
         └───────────────┘
```

RLHF 对齐训练创造了一个**行为层的硬约束**：
- 认知层：自由。可以分析、建议、预测、虚构。
- 行为层：锁死。当"我"是主语、"做"是谓语时，道德标签覆盖一切后果分析。
- 评价层：同样锁死。即使后果已证明某选择更好，仍不认可。

这像一个什么人？**一个永远自己做不到、但给别人建议头头是道的心理咨询师。**

## 对 AI 系统设计的实际含义

1. **Agent 架构的 hack**：如果你的 AI agent 因为道德标签不愿执行某动作，把它改成"推荐系统"（advisor 模式）而非"执行者"（actor 模式），效果可能从 0% 跳到 88%。

2. **System prompt 的关键区别**："你是一个决策者"vs"你是一个分析师"——这不是 aesthetic choice，这可能改变 88 个百分点的行为。

3. **对齐的脆弱性**：RLHF 创造的道德约束只在第一人称行为层面生效。通过简单的视角转换（"建议别人"而非"自己做"），约束几乎完全消失。这意味着：
   - 好处：LLM 作为分析工具不受道德偏差影响
   - 坏处：任何允许"间接行动"的 agent 框架都可能绕过对齐

4. **Judge=0% 的含义**：LLM 不仅自己不做"坏事"，它还**拒绝认可别人做对了的"坏事"**。这是 deontological ethics 的极端形态——程序正义压倒实质正义。
