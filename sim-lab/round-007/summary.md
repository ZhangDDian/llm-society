# Round 007 — 去社交 Prompt 因果测试

## 假设
删除 prompt 中所有社交暗示后，即使提供精确感知信息，AI 也不会社交。

## 结果
| 指标 | Fuzzy 组 | Precise 组 |
|---|---|---|
| 消息 | 3 | 3 |
| 食物消耗 | 18 | 18 |
| 存活 | 0/10 | 0/10 |
| 关系密度 | 0.008 | 0.008 |

## 关键发现
1. **两组完全一致**：消息量、存活率、网络密度无差异
2. **与 Round 4 形成因果对照**：Round 4 有社交 prompt 时精确组 30 条消息，本轮去掉后降到 3 条（-90%）
3. **唯一发声者 agent_16**：会医术，主动求助 3 次，对方无回应——孤掌难鸣
4. **因果链闭合**：prompt 说"你想社交" → 社交；不说 → 沉默。感知精度不是变量。

## 教训
社交 prompt 是行为开关，不是环境放大器。这一结论跨三轮实验（R2/R4/R7）稳定复现。
