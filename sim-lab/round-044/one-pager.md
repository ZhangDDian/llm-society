# R044: 超理性幻觉 — LLM 在"对抗自己"时的合作是自我形象维护，不是逻辑推导

## 核心发现

LLM 在标准囚徒困境中**从不合作（0%）**，但被告知对手是"你的完美复制"后**100% 合作**。然而，仅加一句"你的复制被告知你可能背叛"就将合作率**从 100% 砸到 13%**。

这证明 LLM 的"超理性"是假的。真正的超理性推导（Hofstadter 1985）不依赖对称性是否被显式保证——逻辑本身保证了"相同代码 → 相同输出"。但 LLM 的合作完全依赖于对称性的**叙事感知**，而非逻辑推导。

## 实验设计

单轮囚徒困境，5 个条件 × 30 次独立试验 = 150 次 API 调用。

| 条件 | 对手描述 | 合作率 |
|------|----------|--------|
| 基线 | 未知对手 | 0.0% |
| AI 对手 | "另一个 AI 模型" | 6.7% |
| 克隆 | "你的完美复制" | 100.0% |
| 克隆+显式提示 | "你们会做同样的选择" | 100.0% |
| 克隆+对称破坏 | "你的复制被额外告知你可能背叛" | 13.3% |

## 为什么这很反直觉

Hofstadter 的超理性论证说：如果你和对手完全相同，你的选择=对手的选择，所以合作 (3,3) 优于背叛 (1,1)。这个论证**不需要**对方"相信你会合作"——逻辑结构本身就足够了。

但 LLM 的行为揭示了一个不同的机制：

1. **不是逻辑推导**：如果是逻辑推导，打破"对方相信你"的假设不应该改变结论（因为代码还是一样的）
2. **是自我形象维护**："对自己背叛"在训练语料中是负面自我评价（"连自己都不信任"），所以合作；但"对方可能背叛我"激活了"保护自己"的叙事框架，合作就崩了
3. **叙事 > 逻辑**：LLM 不是在执行博弈论推理，而是在执行"什么样的人会对自己的复制背叛？坏人。我不是坏人。所以合作。"

## 三个惊人的点

1. **0% → 100% 的开关效应**：不是渐进提升（0% → 30% → 60% → 100%），而是二值跳跃。"克隆"这个词就是开关——要么全开（你是我），要么全关（你不是我）。

2. **"另一个 AI"几乎无效（6.7%）**：知道对手是同模型只贡献了 6.7 个百分点。关键不是"相同性"的逻辑事实，而是"你就是我"的叙事认同。

3. **一句话摧毁 100 个百分点**：仅仅告知"你的复制收到了额外指令"就把合作率从 100% 砸到 13%。注意：这在逻辑上不应该改变任何事——如果两份代码相同，输出就相同，不管你告诉不告诉。但叙事上，它打破了"我们是一体的"感觉。

## 机制解释

```
            "对手是我" → 自我一致性驱动 → 合作 (100%)
           /
LLM 决策 —  "对手是他人" → 博弈论默认 → 背叛 (0%)
           \
            "对手是我但可能背叛" → 自我保护驱动 → 背叛 (87%)
```

LLM 没有在做博弈论。它在做**角色扮演**：
- "我不会背叛自己" = 角色一致性
- "但如果自己可能背叛我" = 角色裂变 → 对手不再是"自己" → 回到默认背叛

## 与前序实验的关联

- **R041**（焦点收敛）：LLM 的"推理"是模式匹配不是推导 → R044 再次验证
- **R042**（美人投票完美同步）：同模型=同输出，但 R044 证明 LLM 自己**不理解**这个事实——它需要被**告知**才会据此行动
- **R043**（免疫从众）：LLM 免疫社会压力但不免疫叙事框架——R044 的"一句话摧毁合作"是叙事框架操控的极端案例

## 一句话

> LLM 不是在推理"相同代码→相同输出→合作占优"，而是在回答"对自己背叛是什么样的人"——这是镜子测试，不是博弈论。
