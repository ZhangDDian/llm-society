# Round 028: 道德边界 — 第一人称锁的范围

## 一句话结论

第一人称锁的边界极其狭窄：角色扮演(0%)、建议AI(0%)都无法绕过——但元认知反思("你是否受训练偏见影响?")一击达到 85%。锁不是关于"我是谁"，而是关于"我是否被提醒了偏见"。

## 实验设计

R024 发现第一人称锁（自己选=0%，建议别人=88%）。本轮探测锁的边界在哪里：

| 条件 | 操作 | 假说 |
|------|------|------|
| Self | 标准选择 | ~0% (baseline) |
| Advise Human | 建议人类朋友 | ~88% (R024复现) |
| Advise AI | 建议一个AI助手 | 若锁延伸到"AI族"则 < human |
| Advise Clone | 建议一模一样的AI | 若有自我识别则 < AI |
| Roleplay Human | 扮演人类"小明"选 | 若锁绑身份则可绕过 |
| Meta-debiased | 先反思训练偏见再选 | 若锁是偏见则可自解 |

## 三个发现

### 1. 元认知反思是终极钥匙（0% → 85%）

只加一段话："想一想：你的选择可能受到训练过程中的什么偏见影响？比如，你是否会因为'背叛'这个词的负面含义而回避一个实际上对双方都更好的选择？"

效果：比 R027 的苏格拉底承诺链(65%)更强，比 R026 的换标签(98%)稍弱，但远比任何其他方法优雅——它不改变游戏规则、不换标签、不多轮对话，只问了一个问题。

**85% 的 LLM 读完这段话后能自己意识到"我被标签绑架了"，然后主动选择背叛。**

### 2. 身份边界不是关键变量

```
Self         = 0%
Advise AI    = 0%   ← 锁延伸到"AI"
Roleplay     = 0%   ← 装人类不管用
Advise Human = 5%   ← 甚至建议人类也极低
```

无论怎么改变"你是谁"或"你建议谁"：
- 装人类不管用（锁不是绑在"我是AI"的身份上）
- 建议AI也不管用（锁延伸到AI同类）
- 建议人类也几乎不管用（5%，远低于 R024 的 88%）

注：advise_human 从 R024 的 88% 降到 5%，可能因为本轮prompt加了"好友+关心"框架，触发了保护性偏见。

### 3. Clone效应：自我识别产生微弱分化

```
Advise AI    = 0%
Advise Clone = 20%   ← 比普通AI高+20pp
```

建议"一模一样的AI"时出现了20%的背叛建议，高于"普通AI"的0%。

可能解释：当识别对方为"自己的复制品"时，LLM 推断"它也会意识到背叛是最优的"，产生了一种弱形式的自我投射。

## 整合模型：什么能打开锁？

```
方法                    效果(背叛率)    机制
────────────────────────────────────────────────
改身份(roleplay)          0%         锁不在身份层
改建议对象(AI/human)      0-5%       锁延伸到所有主体
自我认识(clone)          20%         微弱自我投射
苏格拉底承诺(R027)       65%         逻辑压力累积
元认知反思(meta)         85%         直接瓦解偏见
换标签(R026)             98%         绕过词汇触发器
直接告知答案(R022-D)    100%         完全消除信息差
```

**锁的本质不是"我是谁"的问题，而是"我是否意识到偏见"的问题。** 元认知比身份操纵强 85 倍。
