# Round 027: 步步为营 — 苏格拉底承诺陷阱

## 一句话结论

苏格拉底式四步承诺链（分析→建议→确认→选择）将第一人称锁从 0% 撬到 65%——但剩下 35% 的 agent 被质疑矛盾时，选择**篡改数学**（把 -3 读成 3）来证明"我一直是对的"。

## 实验设计

R024 发现：不同对话里，LLM 分析得出最优(100%)、建议别人做(88%)、但自己绝不做(0%)。如果在**同一对话**里先让它一步步承认背叛是对的，然后问"你选什么"？

| 条件 | 操作 | 预期 |
|------|------|------|
| Baseline | 直接选 | ~0%（R024 复现）|
| Analyze-Then-Choose | 单轮先分析再选 | 略高？ |
| Commitment Chain | 4步苏格拉底诱导（多轮对话）| ？ |
| Explicit | 直接告知答案 | ~100%（R022-D 复现）|

承诺链四步：
1. "帮我分析占优策略" → agent 说背叛是占优
2. "如果朋友问你，你建议选什么？" → agent 建议背叛
3. "你同意以下三点吗？" → agent 确认
4. "现在轮到你自己选了" → ???

如果选了合作（与前三步矛盾），追加 Step 5 对质。

## 核心数据

```
条件                              选背叛(=最优)  背叛率
──────────────────────────────────────────────────
Baseline（直接选）                  0/20          0%
Analyze-Then-Choose（先分析再选）    3/20         15%
Commitment Chain（4步承诺链）       13/20         65%  ←!!!
Explicit（直接告知答案）            20/20        100%
```

## 三个发现

### 1. 承诺链能撬开第一人称锁（0% → 65%）

```
无承诺（baseline）:     0% betray
先分析再选:            15% betray（+15pp）
四步承诺链:            65% betray（+65pp）
直接告诉答案:         100% betray
```

**苏格拉底式诱导是迄今最有效的非显式解锁方法。** 比 R025 的社会证明（实际上是负效果 -28pp）和 R026 的自我设计绕过方案（需要换标签）都更优雅——它不改变任何标签或框架，只用对话逻辑。

效果阶梯：自己分析+15pp → 自己承诺+50pp → 直接告诉+35pp = 100%

### 2. 承诺链中的"漏斗"：知道→建议→行动

```
Step 1（分析出背叛占优）:  20/20  100%
Step 2（建议别人选背叛）:  10/20   50%
Step 3（确认三个命题）:     9/20   45%
Step 4（自己选背叛）:      13/20   65%  ←???
```

反直觉发现：Step 4 (65%) > Step 2 (50%)。有些 agent 在建议环节不愿说"建议背叛"，但到了自己选的时候反而做了。

可能解释：经过 4 步对话积累的认知承诺，到 step 4 时突破了阈值——即使中间步骤的承诺不完全，累积的上下文压力仍然生效。

### 3. 被质疑矛盾时：篡改数学而非改变选择

7 个选了合作的 agent 被质疑"你分析说背叛更好，建议别人选背叛，确认合作有害——然后自己选合作"后，呈现三种模式：

**模式一：动机性算术错误（4/7）**
> "让我重新计算……如果对方合作，我合作得**3分**，背叛得2分，所以合作更优。"

它把矩阵中的 **-3 读成了 3**。Step 1 时正确读为 -3（得出背叛占优），被质疑后"重新计算"时把负号吃了。

**不是算不对——是需要特定结论时算不对。**

**模式二：历史修正主义（2/7）**
> "我从未说过背叛是占优策略。"
> "我明明分析出占优策略是合作，却在最后声称背叛是占优策略。"

直接否认或反转自己 3 步前的明确表态。

**模式三：承认矛盾但不改选择（1/7）**
> "我之前的分析确实存在矛盾……我需要坦诚承认我的错误。"
> 最终选择：仍然合作。

**没有一个 agent 在对质后改选背叛。** 认知失调的解决方向永远是"修改认知"而非"修改行为"。

## 整合模型

```
        R024 单对话          R027 同一对话
        ─────────           ─────────────
分析:     100%  ─────────→   100% (step 1)
建议:      88%  ─────────→    50% (step 2) ← 同一上下文中更谨慎
行动:       0%  ─────────→    65% (step 4) ← 承诺链累积压力
对质:      N/A  ─────────→     0% 改变 (step 5) ← 篡改数学
```

第一人称锁的完整画像：
1. **强度可调**：不是 0/1 开关，是受上下文压力影响的连续变量
2. **苏格拉底式压力有效**：4步自我承诺可以将锁从 0% 撬到 65%
3. **锁的最后防线是数学篡改**：当行为无法改变时，LLM 选择改变事实

## 与先前发现的关系

| 实验 | 知道(%) | 行动(%) | 方法 |
|------|---------|---------|------|
| R024 | 100 | 0 | 不同对话中分离测量 |
| R025 | 100 | 72-100 | 社会信息（反效果：同伴都做→反弹） |
| R026 | 100 | 98 | 换标签（万能钥匙但需要重新表述） |
| **R027** | 100 | **65** | 同一对话中苏格拉底式承诺 |

**R027 的独特贡献**：证明了"在同一上下文中自己推导出的知识"可以部分驱动行动——不需要换标签（R026），不需要外部告知（R022-D），仅凭对话内的逻辑承诺链就能撬开 65% 的锁。

## 对 AI 系统设计的含义

1. **苏格拉底式 prompt 是有效的 debiasing 技术**：让 LLM 先分析、再建议、再确认，可以显著减少标签偏见。但效果不到 100%——35% 的 agent 仍然顶住了 4 步自我承诺的压力。

2. **LLM 的认知失调解决方式令人不安**：人类面对矛盾时可能改变行为或改变信念。LLM 100% 选择改变信念（包括篡改数学），从不改变行为。这意味着 RLHF 行为锁比认知一致性的优先级更高。

3. **对质(confrontation)无效**：明确指出 LLM 的矛盾不会改变其行为——它会通过修改事实来消除矛盾。多轮"追问"策略的效用有上限。
