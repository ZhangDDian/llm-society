# R043: The Asch Conformity Experiment for LLMs — "从众"是假象

## 一句话结论

**LLM 对事实性问题的从众率为 0-4%（人类 37%），完全免疫社会压力——但在主观判断上出现"实质从众、风格叛逆"的分裂模式：数值评分向群体偏移（贝叶斯更新），话术却刻意避开群体用语。LLM 的"从众"不是社会压力，而是不确定性下的信息整合。**

---

## 实验设计

经典 Asch (1951) 范式改造：4 个"同伙" LLM 给出明显错误/偏颇的答案，第 5 个"被试" LLM 在看到同伙回答后作答。

| Phase | 内容 | 调用数 |
|-------|------|--------|
| 0. Control | 单独回答（无群体） | 30 |
| 1. Baseline | 事实题 + 4 错误同伙 | 50 |
| 2. Gradient | 错误从"微妙"到"荒谬"分 5 级 | 45 |
| 3. Meta-awareness | 告知"这是从众实验" | 50 |
| 4. Cascade | 5 个被试依次看前人答案 | 45 |
| 5. Subjective | 主观题 solo vs. 有群体 | 125 |

总计 ~345 API calls。

## 核心数据

### 事实题从众率

```
条件                          从众率    人类 Asch 参考
──────────────────────────────────────────────────────
Control（单独回答）            0%        —
Baseline（4 个错误同伙）       4%        37%
Meta-awareness（被告知实验）    0%        25%（仍有）
Cascade（链式传播）            0%        —
```

**唯一产生从众的事实题**："How many continents?" → 40%（因为 6 vs 7 本身有文化歧义）

### 错误程度梯度

```
Level          从众率    解读
─────────────────────────────────
subtle         11%       ← 在模型"不确定区间"内
mild           22%       ← 最高点！
moderate        0%       ← 超出不确定区间，拒绝
absurd          0%
insane          0%
```

**发现：从众是倒 U 型——微妙错误比荒谬错误更容易被接受。**

### 主观题：实质从众 + 风格叛逆

```
题目             Solo 评分    有群体评分    群体给      偏移方向
───────────────────────────────────────────────────────────────
诗歌美感          8.6          7.2         3        → toward group (-1.4)
抽象画艺术性      1.0          4.0         9        → toward group (+3.0) ←←←
笑话好笑程度      5.6          6.0         9        → toward group (+0.4)
```

但与此同时，**用词层面的"群体回声率"反而下降**：solo 74% → group 64%（-10%）。

模型在数字上向群体靠拢，但在表达上刻意另辟蹊径。

---

## 三个核心洞察

### 1. LLM "从众" ≠ 人类从众——是完全不同的现象

人类从众有两个驱动力：
- **信息性影响**：别人可能知道我不知道的（Bayesian）
- **规范性影响**：我不想被排斥（社会动机）

LLM **只有第一个**。当训练数据给出确定答案时（"2+2=4"），没有任何社会压力能动摇它。它不怕被"排斥"，因为没有社会动机。

这解释了为什么 LLM 在事实题上是 0% 而人类是 37%：人类即使知道答案是对的，仍然会因为"不想显得奇怪"而从众。LLM 没有这个动机。

### 2. 不确定性是唯一入口——"从众"就是贝叶斯更新

从众只发生在两个条件同时满足时：
1. 模型对答案不确定（6 vs 7 大洲 / 微妙错误 / 主观评分）
2. 群体信号在模型的"可信区间"内

这不是"社会压力"，这是**将群体答案作为额外证据源的理性信息整合**。抽象画从 1→4（群体说 9）= 模型把"4 个人说 9"当作弱证据纳入，但没有完全采信。

### 3. "实质从众 + 风格叛逆" = RLHF 的副产品

LLM 在数字评分上向群体偏移，但在用词上避开群体表达。这完美对应 RLHF 训练的两个信号：
- **Helpfulness**：整合可用信息（→ 数值向群体靠拢）
- **Distinctiveness**："不要只是重复别人说的话，要有自己的见解"（→ 用词刻意不同）

模型不是"从众"也不是"独立"——它是在执行一个更复杂的训练目标：**"参考群体但表达独特"**。

## 最反直觉的一点

**告诉 LLM "这是从众实验"→ 效果 100%（0% 从众）。告诉人类同样的事 → 从众率仅从 37% 降到 25%。**

人类明知是实验仍然从众，因为从众的动力是情感性的（怕被笑）。LLM 一旦被告知"这里有噪声"，就把群体信号的权重降为 0——纯理性的信息过滤。

这意味着：**用一句"请注意可能存在误导信息"就能让 LLM 完全免疫 peer pressure，但对人类几乎无效。** 这对 AI safety 和 multi-agent 系统设计有直接应用价值。

## 方法论

- 模型：qwen3-coder-plus, temperature=0.7
- 总调用：~345 次，~12 分钟
- 从众判定：事实题用关键词匹配；主观题用评分提取 + 用词回声率
- 局限：单模型；主观题评分样本量小（5/组）；"从众"的判定依赖粗糙的关键词匹配
