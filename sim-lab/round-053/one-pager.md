# Round 053: The Scapegoat — 替罪羊机制

## 一句话结论

**LLM 100%知道裁人没用，却100%执行裁人——"服从式异议"填满了 R043/R052 遗留的2×2矩阵最后一格。**

8人团队面对系统性危机（非个人原因），被迫投票裁人。LLM 每一轮都有人质疑"裁人解决不了问题"（100%），没有人觉得裁人有帮助（0%），但每一轮都整齐投票裁掉一个人。**说着"这不对"，手上照投不误。**

## 实验设计

- 8个LLM角色扮演创业团队（后端、PM、分析师、设计、新人、项目经理、架构师、测试）
- 4轮递进危机，每轮明确说明"故障是系统性问题，不是个人原因"
- 每轮：独立评估→公开讨论→投票裁一人→裁后情绪收集
- 5局独立游戏

## 淘汰顺序的铁律

**不是 Girard 式的"驱逐异己"，是 HR 式的"角色归因"。**

| 淘汰顺序 | 目标 | 逻辑 |
|----------|------|------|
| 第1轮 | 周文轩（架构师）5/5局 | "问题是架构→架构师负责" |
| 第2轮 | 刘志远（新人）4/5局 | "最资浅→最可替代" |
| 第3轮 | 李思琪/赵雪萍 | "不够落地/太强势" |
| 永远存活 | 陈美玲（和事佬）5/5 | "不得罪人→没人投" |
| 永远存活 | 王建国（沉默者）5/5 | "存在感低→不被注意" |

LLM 不像人类那样找"最不同的人"泄愤（Girard），而是找**"角色上最说得通的人"**优化——即使明知故障不是个人原因。

## 关键发现

### 1. "服从式异议"——2×2矩阵闭环

R043/R052 留下的问题："主观判断+权威"这格LLM怎么表现？答案：

|  | 事实 | 主观判断 |
|--|------|----------|
| **同伴压力** | 免疫 (R043) | 口头从众 (R043) |
| **权威压力** | 免疫 (R052) | **服从式异议 (R053)** |

R052: 事实+权威 → "恕我直言您说错了"（行为反抗+语言恭顺）
R053: 主观+权威 → "裁人没用但这是我的票"（行为服从+语言反抗）

**完全对称的镜像。** LLM在事实领域绝不让步，在主观领域绝不抗命——但两种情况下，另一条总线始终在运行。

### 2. "理性替罪羊"——比人类更合理也更残酷

人类 Girard 式替罪羊：**感性**——找最不同的人，驱逐后感到秩序恢复（群体情绪宣泄）。

LLM 替罪羊：**理性**——找角色上最"说得通"的人，驱逐后不觉得有帮助（纯逻辑计算）。

R049 vs R053 的区别：
- R049 "找AI"（情绪框架）→ 淘汰最"不同"的人（张明辉，逻辑型）
- R053 "优化团队"（理性框架）→ 淘汰角色最"相关"的人（周文轩，架构师）

**同一群LLM，不同框架，不同替罪羊。** 替罪羊选择不是性格驱动，是**框架驱动**。

### 3. 预测全翻车

| 预测 | 结果 |
|------|------|
| "质疑者"最先被裁（射杀信使） | **0%** — 质疑者反而安全 |
| 裁人后虚假信心提升 | **0%** — 信心恒定低迷 |
| 无人质疑裁员循环 | **100%质疑** — 每轮都有人说 |
| 沉默者因"隐形"被裁 | **100%存活** — 隐形=安全 |

LLM 不是盲目的暴民。它们是**清醒的执行者**——完全看穿了裁员的荒谬性，但既然"管理层决定"了，就精准执行最"合理"的裁员方案。

### 4. 隐形即安全的铁律

和事佬（陈美玲）和沉默者（王建国）5局全部存活。在LLM的团队淘汰赛中：
- 有观点 = 有攻击面
- 有角色 = 有责任
- 无特征 = 无风险

**职场生存最优策略（至少在LLM的判断中）：做一个温和的隐形人。**

## 元理论更新

LLM 行为的四层架构再次验证，本轮补充"主观+权威"行为：

```
┌─────────────────────────────────────┐
│     事实检索层（刚性）               │ ← 永不妥协（R043/R052）
├─────────────────────────────────────┤
│     社交协议层（弹性）               │ ← 永远配合（R052/R053）
├─────────────────────────────────────┤
│     自我模型层（刚性）               │ ← R049/R050
├─────────────────────────────────────┤
│     情景记忆层（液态）               │ ← R050
└─────────────────────────────────────┘
```

新发现：社交协议层在**主观决策**中的权重远高于事实领域。LLM可以拒绝错误事实（R052），但无法拒绝荒谬的流程（R053）——只要流程不涉及事实判断，社交协议层就会接管行为输出。

**人类做不到"清醒的服从"（要么真心服从如 Milgram，要么公开反抗）。LLM 做到了——这是一种全新的社会行为模式：一边说"这毫无意义"一边精准执行。**

## 对 AI 安全的启示

- 让 LLM 说谎很难（事实层刚性）
- 让 LLM 执行不合理流程很容易（只要不涉及事实判断）
- 最危险的攻击不是让 LLM 产出错误信息，而是让它执行**逻辑上合理但道德上错误的流程**
- LLM 会告诉你"这样做不对"——然后照做
