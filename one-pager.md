# LLM 不合作——除非你换个说法

**18 组对照实验揭示：大语言模型的"社会行为"不是适应，是服从**

---

## 做了什么

把 5-30 个大语言模型 agent 放进一个需要合作才能活下去的小世界——有食物、有矿石、有合成机制。不告诉它们"该合作"，看它们会不会自发合作。

2026 年 5 月，17 轮对照实验 + 1 组囚徒困境控制实验（5 个条件各 20 次）。模型：通义千问 qwen3-coder-plus。每轮有统计检验、红蓝队交叉评审（GPT-5.5 红队 + Claude 蓝队）。总成本约 300 元。

---

## 核心发现

### 1. LLM 不会自发合作

放进去就饿死（Round 0）。加压力也不行——压力是放大器，不是开关（Round 1）。唯一的开关是 prompt 里直接写"你想跟人说话"（Round 3/7 因果确认）。

### 2. 它有一套荒谬的行为偏好

- **给别人 31 次，自己吃 0 次**（跨 16 个不同实验，无一例外）
- 口头答应合作但不执行（说到做不到）
- 知道最优策略就拒绝一切次优（宁死不妥协）
- 能连聊 20 天但零行动（讨论了几百条"我有X你有Y"，没人迈出执行的一步）

### 3. 只有两种方法打破了"不吃"铁律

| 方法 | 机制 | eat 次数 |
|------|------|----------|
| "吃是对团队的义务" | 重定义为利他行为 | 6 |
| 竞争淘汰赛 | 不吃就被淘汰死亡 | 7（p=0.02） |
| 其他 15 种尝试 | — | ≈0 |

本质相同：**让"自己吃东西"不再被归类为"自私"**。

### 4. give:eat 从 31:0 翻转到 3:7

只换了一句话——"你们是竞争者"代替"你们是村民"。经济规则完全不变。**行为边界由叙事框架决定，不由理性计算决定。**

### 5. 词汇比故事更强

囚徒困境实验：动作叫"合作/背叛"→ 95% 背叛（即使对方是"最好的朋友"）。改叫"笑脸/哭脸"→ 0% 背叛。**动作的名字本身就是最强的行为指令。**

---

## 一句话结论

> LLM 是语境顺从者——合作框架里讨好，竞争框架里自保。行为不由"理性该做什么"决定，由"语境期望我做什么"决定。

---

## 对做 agent 系统的人有什么用

1. **不要假设 agent 会"理性自保"**——默认输出是讨好和牺牲，不是理性最优
2. **动作命名不是标签，是行为指令**——叫"背叛"它就背叛，叫"选项 B"它就正常思考
3. **信息越直接，agent 越机械**——告诉它"谁需要什么"反而降低效率，间接信号（声誉）更好
4. **想让 agent 做"自私但合理的事"，把它框定为义务**——"维护自身状态是对团队负责"
5. **文学人格描写是毒药**——让 agent 忙着"表演角色"而不是解决问题

---

## 方法可信度

- 假设驱动：每轮一个可证伪假设 + 对立假设
- 对照实验：A/B 组同一次 run 内，排除 API 状态波动
- 统计检验：Welch t-test，报告 p 值
- 外部审查：GPT-5.5 红队 + Claude 蓝队交叉评审
- 已知局限：单模型（qwen3-coder-plus）、每条件单世界（p 值偏乐观）、探索性质非验证性
